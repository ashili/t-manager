# Project 2 for IS 322

A task manager using ReactJS

### Use:

Download the repo using:

```bash
git clone
```

then:
```bash
cd t-manager
```

then:

```bash
npm install
```

then:

```console
npm start
```

### Frameworks and libraries used:

- React
- Axios
- Material UI
- React beautiful dnd
- My-json

### Purpose:

Getting familiar with the React framework.
