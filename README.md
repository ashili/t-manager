# Project 2 for IS 332

a task manager using ReactJS

### Use:

download the repo using:

```bash
git clone
```

then:

```bash
cd t-manager
```

then:

```bash
npm start
```

### Frameworks and libraries used:

- React
- axios
- Material UI
- react beautiful dnd
- my-json

### Purpose:

getting familiar with the React framework.
